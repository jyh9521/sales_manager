# Sales Manager (販売管理システム)

[中文](README.md) | [日本語](README_JA.md) | [English](README_EN.md)

## 概要

Sales Managerは、ElectronとReactをベースにした最新の販売管理アプリケーションです。中小企業の請求書作成、顧客管理、商品カタログ管理を簡素化するために設計されています。古いツールからの置き換えを目指し、直感的なMaterial Designユーザーインターフェースと自動化機能を提供します。

## 主な機能

*   **請求書管理**:
    *   請求書の作成、編集、表示、削除。
    *   **自動計算**: 小計、消費税（8%と10%の税率に対応）、合計金額を自動計算します。
    *   **入力補完**: 顧客や商品を検索・選択できるオートコンプリート機能を搭載。
    *   **印刷対応**: A4用紙に最適化された印刷ビュー（不要な要素は非表示になります）。
    *   **単位の自動保存**: 新しい単位を入力すると、次回の使用のために自動的に保存されます。
    *   **一括操作**: 請求書の一括削除に対応。
    *   **一括印刷**: 複数の請求書を選択してまとめて印刷可能。

*   **見積書管理**:
    *   見積書の作成、編集、管理。
    *   **ワンクリック変換**: 見積書を直接請求書に変換し、すべての情報を自動的に引き継ぎます。
    *   **印刷対応**: 専用の見積書印刷フォーマット。

*   **顧客管理**:
    *   顧客情報（名称、住所、連絡先）の管理。
    *   特定の顧客の過去の請求書をワンクリックでフィルタリング表示。
    *   郵便番号からの住所自動検索機能。

*   **商品管理**:
    *   単価、税率、関連プロジェクトを含む商品カタログのメンテナンス。
    *   **在庫管理**: 在庫数を自動的に追跡し、請求書発行時に自動的に差し引きます。
    *   請求書作成時に商品情報を自動入力。

*   **設定**:
    *   自社情報のカスタマイズ（請求書のヘッダーに使用）。
    *   会社ロゴのアップロード。
    *   テーマカラーの変更。
    *   消費税率の設定。
    *   **データ保護**: データベースの自動バックアップ、および手動によるバックアップのインポート/エクスポートに対応。

*   **モダンな体験**:
    *   Material UIに基づいたレスポンシブデザイン。
    *   わかりやすいサイドバーナビゲーションとダッシュボード。
    *   完全な多言語対応（日本語、英語、中国語）。

## 技術スタック

*   **フロントエンド**: React 18, TypeScript, Vite
*   **UIフレームワーク**: Material UI (@mui/material), Emotion
*   **デスクトップアプリフレームワーク**: Electron
*   **データベース**: Microsoft Access (.accdb) via `node-adodb`

## インストールと実行

1.  **依存関係のインストール**:
    ```bash
    npm install
    ```

2.  **開発モードの起動**:
    ```bash
    npm run dev
    ```
    これにより、Vite開発サーバーとElectronウィンドウが同時に起動します。

3.  **本番ビルドの作成**:
    ```bash
    npm run build
    ```

## 注意事項

*   このプロジェクトは、データベース接続にWindows環境のMicrosoft Accessドライバーを使用します。システムに適切なODBCドライバーがインストールされていることを確認してください。
*   データベースファイルはプロジェクトのルートディレクトリにある `sales.accdb` です。
